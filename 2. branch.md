
1. branch status
	$ git branch
	* master (* --> on this branch)

2. add branch
	$ git branch newbranch

3. go to branch
	$ git checkout newbranch

4. merge to master(after work on new branch commit it go to master for merge)
	$ git merge newbranch

5. delete file from git an computer
	$ git rm delfile.txt

6. if without commit checkout on master branch
	$ git checkout master
	D       delfile.txt --> delete
	M       modfile.txt --> modify
	Switched to branch 'master'

7. delete branch
	$ git branch -d final
